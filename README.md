# Тестовое задание Angular 2025 - Клиентская система управления пользователями

## Описание проекта

Приложение для управления списком пользователей с возможностью просмотра, добавления, редактирования и удаления записей. Реализовано с использованием Angular и Angular Material.

## Функциональные требования

- Отображение списка пользователей в таблице
- Добавление новых пользователей через модальное окно
- Редактирование существующих пользователей
- Удаление одного/нескольких/всех пользователей
- Валидация полей формы
- Сортировка по колонкам таблицы
- Адаптивный дизайн

## Технические детали

### Используемые технологии

- Angular 16
- Angular Material
- RxJS
- ngx-scrollbar

### Запуск проекта

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm start
```

Приложение будет доступно по адресу `http://localhost:4200`

### Валидация полей

- **Имя**: обязательное, минимум 2 символа
- **Фамилия**: обязательное, минимум 2 символа
- **Email**: обязательное, валидный email
- **Телефон**: необязательное, но должно быть валидным для российских номеров

## Обоснование выбора библиотек

1. **Angular Material** - предоставляет готовые UI-компоненты, соответствующие Material Design
2. **ngx-scrollbar** - кастомные скроллбары с плавной анимацией
3. **RxJS** - для работы с асинхронными операциями и состоянием
